Per cambiare il percorso in cui salvare i file è necessario cambiare la variabile "percorso" in playlist.c. E' possibile anche modificare i tipi di file da gestire con la variabile "estensione". Ricordarsi di cambiare il percorso anche in "pulisci_percorso()"

Per aggiungere brani è sufficiente togliere i commenti alle istruzioni nel main e modificare i dati; Tutti i dati saranno sovrascritti.
Aprire il file con "a" per non sovrascrivere.
